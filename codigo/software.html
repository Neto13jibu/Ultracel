¿Qué es software?

Programa o conjunto de programas de cómputo que incluye datos, procedimientos y pautas que permiten realizar distintas tareas en un sistema informático.

Comúnmente se utiliza este término para referirse a los programas que permiten el funcionamiento de un dispositivo, en este caso de un celular.

Desbloqueo de patron y pin
